1. Установка и настройка ОС

  • Настрой параметры ОС: sudo dpkg-reconfigure locales, timedatectl
  • Установи драйверы: sudo ubuntu-drivers autoinstall
  • Настрой интернет: nmcli dev wifi connect
  • Проверь соединение: ping 8.8.8.8
  • Установи ПО: LibreOffice, GIMP, архиватор
  • Установи CUPS и виртуальный принтер: sudo apt install cups

2. Защита компьютерных систем

  • Резервное копирование: timeshift --create
  • Создание ISO: Clonezilla или Deja Dup
  • Добавить пользователей: sudo adduser testuser
  • Права: chmod, chown
  • Аутентификация: PAM, passwd
  • Журнал: journalctl -xe, logrotate
  • ОТ КОЛЯДИНА: копирование tar cvpzf backup.tgz –exclude=/proc –exclude=/lost+found –exclude=/backup.tgz –exclude=/mnt –exclude=/sys –exclude=/web /

4. Документирование ПО

Согласно ГОСТ Р 59795–2021, документация пользователя должна включать:

  • Наименование программного обеспечения
  • Назначение программы — краткое описание, зачем она нужна
  • Способы запуска — как и где запустить программу
  • Функциональные возможности — список задач, которые выполняет ПО
  • Основные действия пользователя — что и как делать в интерфейсе
  • Способы завершения работы — как правильно выйти

Также следует оформить руководство по использованию: описать интерфейс, меню, кнопки и примеры сценариев работы.

Пример оформления:

Наименование: GIMP (GNU Image Manipulation Program)
Назначение: Редактор изображений для обработки и создания графики
Запуск: Программы → Графика → GIMP
Функции:
- Открытие и редактирование изображений
- Использование слоёв
- Работа с цветами, фильтрами и масками
- Экспорт в различные форматы (PNG, JPG, PSD)
Интерфейс: Главное меню, панель инструментов, окно слоёв
Завершение: Файл → Выход

6. Критерии оценивания и действия

  • Настройка ядра (2.5): Открыть файл /etc/sysctl.conf, внести параметры, например vm.swappiness=10. Применить командой sudo sysctl -p.
  • Настройка SSH (2.0): sudo apt install openssh-server, включить sudo systemctl enable --now ssh. Проверить ssh localhost.
  • Удалённый доступ (2.0): Настроить SSH + разрешить через брандмауэр sudo ufw allow ssh.
  • Выбор сетевого интерфейса (1.0): Использовать ip a или nmcli device status и подключить актуальный.
  • Проверка соединения (2.0): Выполнить ping 8.8.8.8 и curl ya.ru.
  • Базовое ПО (2.0): Установить LibreOffice, архиватор: sudo apt install libreoffice p7zip-full.
  • Виртуальный принтер (2.0): Установить CUPS: sudo apt install cups. Перейти в http://localhost:631 и добавить PDF-принтер.
  • Обоснование выбора ПО (2.0): Указать цели, назначение, совместимость — включено в документацию и сайт.
  • Резервное копирование (2.0): Установить Timeshift, создать снимок: sudo timeshift --create --comments "Initial backup"
  • Образ системы (1.0): Использовать Deja Dup (GUI) или Clonezilla (Live ISO).
  • Точки восстановления (2.0): Timeshift → "Create" (через GUI) или через терминал.
  • Создание пользователей и групп (1.0): sudo adduser analyst, sudo groupadd team, sudo usermod -aG team analyst
  • Права доступа (1.0): Пример: sudo chown analyst:team /opt/data, chmod 770 /opt/data
  • Аутентификация (1.0): Установить требования к паролю: /etc/pam.d/common-password, изменить пароль passwd user
  • Журналы мониторинга (1.0): Просмотр логов: journalctl -xe. Настройка: /etc/logrotate.conf
  • Установка и настройка ПО (2.5 + 2.0): Установить Android Studio: sudo snap install android-studio --classic. Открыть, настроить интерфейс, цветовую схему, SDK.
  • Обмен данными (2.0): Установить MySQL, phpMyAdmin. Проверить соединение с базой из IDE.
  • Совместимость (5 × 0.5):
    • Цветовая палитра: Настройка темы в настройках системы
    • Низкое разрешение: переключить режим экрана, масштаб 100%
    • Меню и кнопки: настройка DPI через gnome-tweaks
    • Отключить композицию: compton --backend glx или xfwm4 --replace
    • Отключить масштабирование: Параметры экрана → масштаб 100%
  • Документация (2.5): Файл с описанием: название ПО, запуск, функции.
  • Руководство (4.0): Подробное руководство со скриншотами, описанием сценариев
